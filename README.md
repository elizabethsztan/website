# Editing Your Website

## Adding a Section

Add this inside `<main class="wrap">`:

```html
<section>
  <h2>☆ Section Title ☆</h2>
  <p>Your content here.</p>
</section>
```

## Adding a Header

```html
<h2>☆ Big Header ☆</h2>
<h3>Smaller Header</h3>
```

## Adding a Link

```html
<a href="https://example.com">Link Text</a>
```

## Adding an Image

```html
<img src="image.png" alt="description">
```

## Adding a List

```html
<ul>
  <li>Item one</li>
  <li>Item two</li>
</ul>
```

## Adding Decorative Text

Use these anywhere:
- `<span class="blink">blinking text</span>`
- Sparkles: `✧･ﾟ: *✧･ﾟ:*`
- Stars: `☆ ★ ✦ ✧`

## Adding a Box (like the main title box)

```html
<div class="main-box">
  <p>Content in a glowing pink box</p>
</div>
```
